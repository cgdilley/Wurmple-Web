Welcome to Wurmple, the Pokémon Wordle!

The goal of the game is to guess the correct sequence of 5 Pokémon.

Each Pokémon in the sequence must share at least one of its types with the Pokémon preceding it in the sequence.

Any guesses you make will be considered correct as long as its typing exactly matches that of the original answer.

Note: A 'typing' refers to the set of 1 or 2 types belonging to a particular Pokémon.

Hints:

  • An exact typing match in this position; a correct guess
  • An exact typing match to a typing in a different position
  • An exact typing match to a typing in a different position. In addition, one of these types are correct for this position as well
  • One of the types in this position are correct
  • One of the types in this position are found elsewhere in the sequence
  • None of the types match anything in the sequence
